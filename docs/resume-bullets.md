# Resume Bullets

- Built local self-hosted GitLab DevSecOps lab with GitLab CE and registered Docker runners for CI/CD execution across lint, test, security, build, and deployment stages.
- Designed GitLab CI/CD pipeline for Python/Node backend services with Ruff, pytest, npm tests, dependency audits, Docker image builds, and manual production approvals.
- Deployed backend services to local Kubernetes using kind and Helm charts with ConfigMaps, Secrets, Ingress, health probes, resource limits, and environment-specific values for dev/staging/prod.
- Modeled infrastructure-as-code workflows using Terraform and Pulumi for local Kubernetes resources and documented how the same pattern maps to AWS/GCP.
- Integrated DevSecOps checks including dependency audits, secret scanning, container scanning, static analysis, and custom API exposure validation.
- Built Prometheus/Grafana observability dashboards for latency, error rate, request volume, worker health, payment workflow metrics, and severity routing metrics.
- Implemented backup/restore runbook for GitLab self-managed operations and documented disaster recovery steps.
